50条常用Linux命令

发表于 2017-09-13 | 分类于 Linux | 阅读次数

1.pwd 当前工作目录

2.cd 切换工作目录 cd..返回上一级目录

3.ls 显示目标工作目录列表 -l 列出当前目录详细文件信息 -a显示隐藏文件 -lh人性化展示文件列表

4.vim .test创建一个隐藏文件 i 插入内容 保存文件 esc+shift+： 然后wq+enter

5.cat 查看文件内容

6.more 慢慢查看文件内容

7.less:与 more 类似，但是比 more 更好的是，他可以往前翻页

8.mkdir 创建目录

9.mv(选项)(参数) 对文件或目录重新命名，或者将文件从一个目录移到另一个目录中。

10.mkdir -p cn/gansu/beijing/sjdjs/sds 递归创建目录

##

11.tree cn 显示cn目录树结构

12.install 的作用是安装或升级软件或备份数据，它的使用权限是所有用户。

13.zip 可以用来解压缩文件，或者对文件进行打包操作。zip是个使用广泛的压缩程序，文件经它压缩后会另外产生具有“.zip”扩展名的压缩文件。

14.rmdir 用来删除空目录。当目录不再被使用时，或者磁盘空间已到达使用限定值，就需要删除失去使用价值的目录.还可以用带有-r选项的rm命令递归删除一个目录中的所有文件和该目录本身，但是这样做存在很大的危险性。

15.rm 可以删除一个目录中的一个或多个文件或目录，也可以将某个目录及其下属的所有文件及其子目录均删除掉。对于链接文件，只是删除整个链接文件，而原有文件保持不变 ##。

16.dirs 显示当前目录栈中的所有记录（不带参数的dirs命令显示当前目录栈中的记录）。dirs始终显示当然目录, 再是堆栈中的内容；即使目录堆栈为空, dirs命令仍然只显示当然目录。

17.cp 用来将一个或多个源文件或者目录复制到指定的目的文件或目录

18.shutdown 命令用来系统关机命令。

19.reboot,halt,poweroff 命令用来重新启动正在运行的Linux操作系统

20.df 显示磁盘分区上的可使用的磁盘空间 -h

21.top 查看系统性能 q退出

22.ifconfig 显示系统的IP配置

23.logout 注销

24.touch命令有两个功能：一是用于把已存在文件的时间标签更新为系统当前的时间（默认方式），它们的数据将原封不动地保留下来；二是用来创建新的空文件。

25.find / -name file1 从 ‘/‘ 开始进入根文件系统搜索文件和目录

26.find / -user user1 搜索属于用户 ‘user1’ 的文件和目录

27.tar:压缩文件和读取压缩文件

28.cal 2007 显示2007年的日历表

29.date 041217002007.00 设置日期和时间 - 月日时分年.秒

30.clock -w 将时间修改保存到 BIOS

31.uname -r 显示正在使用的内核版本

32.popd 用于删除目录栈中的记录；如果popd命令不加任何参数，则会先删除目录栈最上面的记录，然后切换到删除过后的目录栈中的最上面的目录。

33.head 输出文件的头几行 默认前10行

34.tail 输出文件的后几行 默认前10行

35.diff/vimdiff 比较文件内容

36.free 显示Linux系统中空闲的、已用的物理内存及swap内存,及被内核使用的buffer

37.hostname 设置主机名

38.file 查看文件的类型

39.useradd 添加用户

40.chmod 改变文件/目录属性 （只有文件属主和root 可以修改）

41.chown 改变文件/目录 属主/属组

42.kill +pid 杀死进程

43.exit 退出登录的linux 也可以用ctrl +d 快捷键代替

44.clear 清屏 也可以用ctrl +l 快捷键代替

45.ntpdate 通过网络校准时间

46.alias 设置命令别名 重启后失效

47.unalia取消别名

48.rpm 管理 安装 升级 删除软件包

49.history 查看历史输入的命令纪录

50.ps 显示进程信息
