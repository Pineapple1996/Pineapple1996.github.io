hexo+github搭建个人博客

发表于 2017-07-25 | 分类于 搭建博客 | 阅读次数

1、前言

过完大三的一年，上完了大学所有要在课堂上的课。最近有了大量的时间来投入个人的学习和找工作，所以想要开始写博客了，一来为了整理自己学习到的知识，二来也是为了求职升学所用。而且在我的博客中还可以分享一些自己有趣的小事。
开始在网上看了很多的博客搭建，开始在知乎上看到用WorldPress，我也尝试自己去买了个主机（在这里向大家推荐这个VPS，很便宜一主机），装了个WP搭建的博客。不过在后面的寻找中无意间看到了HEXO+github搭建的方法，框架很好，关键是免费（嘿嘿~）。
顺便将搭建过程记录下来，成为我博客第一篇正式技术博文。为我博客文章充充数，哈哈~
我的博客地址：jiangmoran.top,欢迎大家指导学习。

2、准备

其实下面的准备你只需要安装就行了，按着博文做下去，如果你有兴趣可以看看全部推荐的东西

2.1下载安装git

git是一个分布式的版本控制工具，也是目前最流行的。它可以帮助你管理本地仓库，并且把本地的项目托管到远端的Github上。也就是你需要建立一个仓库来放你的博客网站项目，并部署到Github。
这里推荐廖雪峰的Git教程，或者视频版git教程，不需要看完，只需要看安装的部分，几分钟就搞定。如果有什么问题可以看看视频教程，讲得很清楚。

2.2下载安装Node.js

Node.js是一个Javascript运行环境（runtime）。实际上它是对Google V8引擎进行了封装。Node.js主要用于安装hexo，npm开头的命令都依赖于Node.js。如果出现以下提示则代表你的Node.js没有安装或者还未生效，如果已经安装了则重启电脑，如果没安装则安装再重启电脑。

bash: nmp: command not found
